◆ CP_StatusTypeDef

enum CP_StatusTypeDef

Statuswerte.

Wird zweierlei verwendet. Zum einen als Rueckgabe-Wert der Anwenderfunktionen, ob diese erfolgreich ausgefuehrt wurden. Zum Zweiten werden die Werte fuer die Steuerungsfelder Status verwendet.
